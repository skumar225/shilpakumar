Keep Calm and Write On
Keep Calm and Write On is an interactive speed writing game, where two players take turns to create a story with only 30 seconds on each turn to write a paragraph to build a story.

Technologies used: Javascript, jQuery, Bootstrap, HTML, CSS
MusicBucket
MusicBucket allows users to categorize their favorite Spotify tracks by moods and organizing these moods in playlists (buckets) built in the application. Users are required to be authenticated by Facebook in order to create their personalized playlists.

Technologies used: Ruby, Rails, Bcrypt, Spotify API, AJAX, PostgresSQL, Javascript, jQuery, Bootstrap, HTML, CSS
OpenMic
OpenMic is a chat application where users can upload audio in real-time, using the SoundCloud API. This allows for easy collaboration on music projects between musicians.

Technologies used: Node.js, Express, Socket.io, Redis, SoundCloud API, Bootstrap, HTML, CSS
